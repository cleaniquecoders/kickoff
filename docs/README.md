## About Kickoff

Kickoff is a Composer package that bootstraps new Laravel projects with opinionated
conventions, packages, and project structure. It automates the setup of development
tools, testing frameworks, deployment scripts, and standardized project architecture.

For usage instructions, see the [main README](../README.md).
